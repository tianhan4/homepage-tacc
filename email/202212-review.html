Features Released in 2022
 
TACC (Turing AI Computing Cloud) has released several new features in year of 2022:
  • May 2022: Persistent storage and incremental updates for user dependencies in the user interface, tcloud. This can speed up the initialization stage in task processing. Users can use the **`tcloud env ls`** and **`tcloud env ls -n`** commands to list and query their stored environments and packages, respectively.
  • July 2022: A legacy file expiration policy that will remove old files stored on TACC on a 30-day rolling window basis. Users should download and re-upload files when needed. This policy will be enforced 24 hours after the announcement. TACC is also scaling up its storage space.
  • November 2022: A web-based GUI console for TACC that allows real-time monitoring and statistics, as well as self-service public key updates. Users can login through the console to monitor tasks, update account information, and query current and historical task status. They can also update their ssh public keys online.
Learn more at tacc.ust.hk.
 
Fields of Future Improvements
 
The TACC team is actively exploring the opportunities to improve TACC in the following fields:
  1. Hardware: TACC may upgrade its hardware components to improve performance, such as by using faster processors, more memory, or faster storage systems.
  2. Software: TACC may improve the software tools and libraries available to users, such as by adding support for new frameworks or by improving the performance of existing tools.
  3. Workflow management: TACC may improve the tools and systems used to manage and schedule tasks, such as by adding support for more complex workflow patterns.
  4. Data management: TACC may improve the tools and systems used to manage and access large data sets, such as by adding support for new data formats.
  5. User experience: TACC may improve the user experience in various ways, such as by making it easier for users to access resources, by providing better documentation and new tools to the user interface.
We welcome all suggestions from our users (you)!
